# AGENTS.md

## Agent skills

### Issue トラッカー

issue はローカル markdown（`.scratch/<機能名>/` 配下のファイル）で管理します。詳細は `docs/agents/issue-tracker.md` を参照。

### トリアージラベル

5つの標準ラベル（needs-triage / needs-info / ready-for-agent / ready-for-human / wontfix）をそのまま使用。詳細は `docs/agents/triage-labels.md` を参照。

### ドメインドキュメント

単一コンテキスト構成（ルートに `CONTEXT.md` + `docs/adr/`）。詳細は `docs/agents/domain.md` を参照。
