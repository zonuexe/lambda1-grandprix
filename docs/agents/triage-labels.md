# トリアージラベル

skill は5つの標準的なトリアージ役割で会話します。このファイルは、それらの役割をこのリポジトリの issue トラッカーで実際に使うラベル文字列に対応づけます。

| mattpocock/skills のラベル | このトラッカーのラベル | 意味 |
| -------------------------- | ---------------------- | ---- |
| `needs-triage`             | `needs-triage`         | メンテナが評価する必要あり |
| `needs-info`               | `needs-info`           | 報告者からの追加情報待ち |
| `ready-for-agent`          | `ready-for-agent`      | 完全に仕様化済み・AFK エージェント着手可 |
| `ready-for-human`          | `ready-for-human`      | 人間による実装が必要 |
| `wontfix`                  | `wontfix`              | 対応しない |

skill が役割に言及したとき（例:「AFK 着手可のトリアージラベルを付与する」）は、この表の対応するラベル文字列を使用します。

実際に使う語彙に合わせて右列を編集してください。
